Inline Styles

An inline style may be used to apply a unique style for a single element.

Disadvantages

  • Inline styles make the HTML code messy and harder to read because mixing of styling instructions right into the HTML tags.

  • If we want to change the look of the website, it's not easy with inline styles. You have to go to each HTML tag and change the style individually.

  • We can't reuse styles easily. If you want two things to look the same, you have to copy the style to both, making your code longer and more prone to mistakes.
